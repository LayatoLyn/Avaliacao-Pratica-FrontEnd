Portfolio profissional

Objetivo deste projeto foi cirar um site funcional simuilando o portfolio de um artista contendo 4 paginas com estruturação semântica em html, estilização em css
e implementação de validação de campos do formulário de contato e manipulação de elementos na página com javascript com mensagens de retorno de erro ou sucesso.

Foram usados somente javascript, css e html usando media query somente no arquivo portfolio.css

A estrutura do site segue a seguinte:

index.html - Pagina Inicial/ Home page
sobre.html - Pagina sobre (mostra informações sobre o artista)
portfolio.html - Pagina do portfolio (mostra as obras relaizas pelo artista)
contato.html - Pagina de contato (para simular um envio de email comercial ao artista, utilizando a validação em javascript e um databank simulado no console)

As funcionalidades implementadas no site são:
-Galeria de imagens da pagina portfolio
-Layout responsivel das imagens da pagina portfolio
-Seção sobre contendo informações sobre vida, carreira e habilidades do porfissional
-Formulário funcional com validação e retorno de sucesso ou erro na pagina contato
